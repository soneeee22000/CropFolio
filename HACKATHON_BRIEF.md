## 🧠 Our Team Strategy (Working Notes)

### Stack Considerations
- **AI/ML:** Python, scikit-learn, PyTorch / TensorFlow, HuggingFace
- **Backend:** FastAPI or Flask
- **Frontend:** React + Tailwind (lightweight, fast to build)
- **Data:** Open climate datasets (NASA POWER, FAO GAEZ, Open Meteo)
- **Deployment:** Vercel / Render / Railway for demo-ready MVP

### Winning Formula
1. **Tight problem scoping** — one specific pain point, solved well
2. **Real data** — use actual Myanmar agri/climate datasets where possible
3. **Demo-ready MVP** — working prototype > polished slides
4. **Impact narrative** — frame around farmers, not just tech
5. **Business viability** — show a path beyond the hackathon

### Ideas to Explore
- [ ] Crop disease detection via mobile photo upload (Vision AI)
- [ ] Weather-based planting calendar / SMS alert system
- [ ] Price prediction for key crops (rice, pulses) using historical data
- [ ] Micro-insurance eligibility scoring using climate risk data
